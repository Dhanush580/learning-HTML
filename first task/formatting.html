formatting a page

every webpage consists of three main sections, they are header, main, footer sections
Header tag is used to define header section in our webpage, the items like navigation menu, logos, search bar, etc are places here in this header section we can place "nav" tag in this header section to display our navbar
this is navbar, all navigation links, lists, buttons go here
main tag is used to define the main section of our webpage, all main content goes into this, like sections, images, etc. we can use many tags in main section like article, section, aside, etc.
this is a article
this is another article
this is a section in article
this is another section in article
this is aside content, can be used to display ads in your webpage
footer tag is used to define the footer section of the webpage. usually footer section consists of social media links, legal rights of the company, other navigation links.

these tags increase the understandability of our website, helps in search engine optimization.
